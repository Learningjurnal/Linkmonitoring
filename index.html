Jalankan SQL ini di Supabase → SQL Editor (ganti linkmon_kv jika Anda ubah nama tabel di atas):
CREATE TABLE linkmon_kv (key TEXT PRIMARY KEY, value TEXT, updated_at TIMESTAMPTZ DEFAULT NOW()); ALTER TABLE linkmon_kv ENABLE ROW LEVEL SECURITY; CREATE POLICY "anon_all" ON linkmon_kv FOR ALL USING (true) WITH CHECK (true); Tabel ini terpisah dari data lain di project Anda — aman dipakai bersama project yang sudah ada. Update Link

Upload Excel / CSV / TXT atau tempel catatan untuk memperbarui link.

Klik atau seret file ke sini .xlsx, .xls, .csv, .txt — header otomatis dipetakan
atau tempel note
Mode pembaruan
MergeCocokkan via Link: update yg ada, tambah yg baru
AppendTambah semua sebagai baris baru
ReplaceHapus semua, ganti dgn file ini
Kunci pencocokan = kolom Link. Header dideteksi otomatis (Name, Link, Status, Region, Output/Part, Update, Last Page). Pastikan nama header file Anda sesuai agar pemetaan akurat.

Edit Link
